# Tic Tac Toe game

It's simple, client-server (via INET-stream-sockets) game with gui-client on GTK2.


## How to play


### Local

1. Clone repo
2. Start server
3. Start client
4. Enjoy.


### With friends

1. Clone repo
2. Player one start a server on some *port* (default - 64501)
3. Player two start a client with args `--host <player 1 IP> --port PORT`
4. Enjoy
